# Chrome Web Store Listing — Notara

## Name
Notara

## Short Description (132 chars max)
Floating sticky notes on any website. Create, color-code, and organize notes right where you need them. Screenshot capture included.

## Detailed Description

Notara lets you place floating sticky notes on any webpage — so your thoughts stay exactly where they matter.

**Key Features:**

- **Floating Notes on Any Page** — Create sticky notes that live right on the websites you visit. They're there every time you come back.
- **Color-Coded Organization** — Pick from multiple note colors to categorize and prioritize at a glance.
- **Screenshot Capture** — Snap a screenshot of any page section and attach it to a note for visual context.
- **Alert Reminders** — Schedule notifications on any note so you never forget to follow up.
- **Right-Click to Add** — Quickly create a note from the right-click context menu.
- **Page or Global Notes** — Choose whether a note appears only on one page or across every site.
- **Sleek Dark Theme** — A clean, distraction-free interface with neon cyan accents that stays out of your way.
- **Google Sign-In** — Sign in to personalize your experience (sync coming soon).
- **100% Local Storage** — Your notes are saved in your browser. Nothing is sent to external servers.

Whether you're researching, studying, project managing, or just bookmarking ideas, Notara keeps your notes right where you need them.

## Permission Justifications

| Permission | Justification |
|---|---|
| `storage` | Required to save your sticky notes locally in the browser so they persist across sessions. |
| `activeTab` | Required to create and display notes on the page you are currently viewing. |
| `contextMenus` | Provides a right-click "Add note" option for quickly creating notes on any page. |
| `alarms` | Used to schedule alert reminders that you set on individual notes. |
| `notifications` | Used to display browser notifications when a note's scheduled alert fires. |
| `identity` | Enables Google sign-in for account authentication. |
| `<all_urls>` (host permission) | Required to display your saved notes on any website you visit. |
